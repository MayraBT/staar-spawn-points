📍 FINDING THE SCALE FACTOR

STEP 1: Read the vertices table

Triangle ABC is dilated with the origin as the center of dilation.

Vertices of △ABC Vertices of △A'B'C'
(2, 3) (6, 9)
(4, 1) (12, 3)
(2, -1) (6, -3)
📝 YOUR TASK: Compare the original vertices (left column) with the dilated vertices (right column). Notice how the numbers changed!

STEP 2: Use Desmos to find the scale factor

Let's use the first vertex: (2, 3) → (6, 9)

🔍 KEY STRATEGY: Divide the NEW coordinate by the ORIGINAL coordinate.
📊 Use the DESMOS calculator on the right →
  1. Type: 6/2 to find x scale factor
  2. Type: 9/3 to find y scale factor
  3. Both should equal 3
Scale factor = new ÷ original = 3

STEP 3: Write the dilation rule

The scale factor is 3, so multiply both x and y by 3.

(x, y) → (3x, 3y)

✅ YOUR TURN - Which rule represents this dilation?
