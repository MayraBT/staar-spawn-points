📍 TEKS 8.10(C) - TRANSFORMATIONS

Triangle ABC is transformed, creating Triangle A'B'C'.

Which rule describes the transformation applied to triangle ABC to create triangle A'B'C'?

🎮 GUIDED WALKTHROUGH

Follow each step using Desmos →

STEP 1: Create Table for Original Points

In Desmos, click the + button and add a table

Enter these points in the table:

x₁: 2, 5, 4

y₁: 3, 3, 6

Then type: polygon(x₁, y₁)

STEP 2: Create Table for Transformed Points

Add another table for the transformed triangle:

Enter these points in the new table:

x₂: -2, 1, 0

y₂: 5, 5, 8

Then type: polygon(x₂, y₂)

💡 You should see both triangles now!

STEP 3: Compare ONE Vertex

Let's use point A and A' to find the pattern:

A(2, 3) → A'(-2, 5)

x-coordinate: 2 → -2 (changed by -4)

y-coordinate: 3 → 5 (changed by +2)

💡 The rule might be: (x, y) → (x - 4, y + 2)

STEP 4: Verify with Another Point

Check if the rule works for point B:

B(5, 3) → B'(?, ?)

Using rule (x - 4, y + 2):

x: 5 - 4 = 1 ✓

y: 3 + 2 = 5 ✓

B'(1, 5) ← Matches!

FINAL QUESTION:

Which rule describes the transformation from triangle ABC to triangle A'B'C'?

✓ PERFECT!

(x, y) → (x - 4, y + 2)
