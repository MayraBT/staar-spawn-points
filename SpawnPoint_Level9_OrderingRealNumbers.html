📍 TEKS 8.2(D) - ORDER REAL NUMBERS

Which list shows the values ordered from LEAST to GREATEST?

7.42     38/5     √61     745%
⚠️ IMPORTANT: Always check if the question asks for LEAST to GREATEST or GREATEST to LEAST!

🎮 GUIDED WALKTHROUGH

We'll plot each value on the x-axis to see their order

STEP 1: Plot First Value (7.42)

In Desmos, type: (7.42, 0)

This plots the point on the x-axis

💡 Color & Label: Click the point → Change color → Enable label → Type "7.42"

STEP 2: Convert & Plot Fraction (38/5)

In Desmos, type: (38/5, 0)

Desmos will show it equals 7.6

Color it differently and label it "38/5"

💡 The label can't show fractions with the "/" symbol, so just type "38/5" in the label box

STEP 3: Convert & Plot Square Root (√61)

In Desmos, type: (sqrt(61), 0)

Desmos will show it's about 7.81

Color it and label it "sq rt 61"

💡 Labels can't show the √ symbol, so write "sq rt 61" or "sqrt 61"

STEP 4: Convert & Plot Percent (745%)

First convert: 745% = 745 ÷ 100 = 7.45

In Desmos, type: (745/100, 0)

Color it and label it "745%"

💡 You can type "745%" in the label (% symbol works!)

STEP 5: Read from Left to Right

Look at your graph! The points from LEFT to RIGHT are in order from LEAST to GREATEST.

The order should be:

7.42 → 745% → 38/5 → √61

(7.42 → 7.45 → 7.6 → 7.81)

FINAL QUESTION:

Which list shows the values ordered from LEAST to GREATEST?

✓ PERFECT!

D) 7.42, 745%, 38/5, √61

7.42 < 7.45 < 7.6 < 7.81
