📍 TEKS 8.9(A) - SYSTEMS OF EQUATIONS

Two lines are represented by these points:

Line 1: (0, 7) and (8, 3)

Line 2: (2, 0) and (0, -5)

Which ordered pair is the solution of the system of equations?

💡 Note: Some problems may provide a graph with two intersecting lines instead of points. The solution (intersection) will NOT be plotted on the graph—you'll need to find it!

🎮 GUIDED WALKTHROUGH

Use Linear Regression to find where the lines intersect

STEP 1: Create Table for First Line

In Desmos, click the + button and add a table

Enter two points from the blue line:

x₁: 0, 8

y₁: 7, 3

💡 Use the labeled points on the graph!

STEP 2: Add Linear Regression for First Line

In the blue pane on the left, click the regression button

It will automatically calculate the equation!

You should see: y = -0.5x + 7

💡 The regression button is in the left margin of Desmos. It looks like a trend line icon.

STEP 3: Create Table for Second Line

Add another table for the orange/red line:

x₂: 2, 0

y₂: 0, -5

Use the labeled points from the second line

STEP 4: Add Linear Regression for Second Line

Click the regression button in the blue left pane again

You should see: y = 2.5x - 5

Now you have both equations!

STEP 5: Find the Intersection Point

Look at your graph! Where do the two lines cross?

The intersection point is the SOLUTION to the system!

💡 You can click on the intersection point in Desmos to see the exact coordinates

FINAL QUESTION:

Which ordered pair is the solution of the system of equations?

✓ PERFECT!

D) (4, 5)

The intersection point is where both equations are true!
